# PTLURLProtocol

## Description
A helpful wrapper for testing network requests using NSURLProtocol.

## Usage
- Subclass PTLURLProtocol
- Override 3 methods
- Register your custom protocol

        [MyCustomURLProtocol registerProtocol];
    
- Profit
